# Lista de Exercícios 01

## Operadores, Entrada e Saída

Esta lista contém exercícios básicos de programação focados em **operações aritméticas, entrada e saída de dados**.

---

## Q1

Faça um programa para ler um número inteiro e imprimir o **dobro** desse número.

---

## Q2

Faça um programa para ler um número inteiro e imprimir o **quadrado** desse número.

---

## Q3

Faça um programa para ler **dois números inteiros** e imprimir a **multiplicação** desses dois números.

---

## Q4

Faça um programa para ler **duas notas de um aluno do IFRN** em um curso semestral.
O programa deverá apresentar a **média desse aluno após as duas provas**.

---

## Q5

Faça um programa que leia um **número inteiro entre 0 e 999** e imprima a **soma dos algarismos** desse número.

Exemplo:

358 gera uma saída **16**, pois:

3 + 5 + 8 = 16

---

## Q6

Faça um programa que leia um **número inteiro entre 0 e 999** e imprima esse número com **3 algarismos**.

Exemplos:

23 gera uma saída
023

8 gera uma saída
008

---

## Q7

Faça um programa que leia um **número entre 0 e 9999** e imprima **cada algarismo em uma linha diferente**.

Exemplo:

Entrada:
3219

Saída:

```
9
1
2
3
```

Entrada:
123

Saída:

```
3
2
1
0
```

---

## Q8

Faça um programa que leia **três números inteiros**, representando a duração de um experimento científico em:

* horas
* minutos
* segundos

O programa deve informar **essa duração total em segundos**.

---

## Q9

Faça um programa que leia um **número inteiro representando segundos** de duração de um experimento científico e imprima o tempo no formato:

h:m:s

Exemplo:

3755 gera uma saída

```
1:2:35
```

---

## Q10

Faça um programa que leia um **caractere minúsculo** e imprima o seu **equivalente maiúsculo**.

---

## Q11

Faça um programa que leia **um inteiro entre 100 e 999** e o imprima **de trás para frente**.

Exemplo:

384 gera uma saída

```
483
```

---

## Q12

Faça um programa que calcule a **média ponderada de três números inteiros**, com pesos:

* 3
* 9
* 15

**Sem usar a operação de multiplicação.**

---

## Q13

Faça um programa que imprima a frase:

```
Hello World
```

---

## Q14

Faça um programa que leia **um número inteiro** e imprima:

* **1** se o valor for **ímpar**
* **0** se o valor for **par**

---

## Q15 

Faça um programa que leia **um número inteiro entre 0 e 9999** e imprima esse número com **4 caracteres**, substituindo o algarismo **0 por espaço**.

Exemplo:

304 gera uma saída:

```
3 4
```

---

## Q16 

Faça um programa que leia **um número inteiro** e imprima:

* **-1** se o valor for **ímpar**
* **0** se o valor for **par**

---

## Q17 

Faça um programa que leia **dois números inteiros** e calcule a **média aritmética simples**, apresentando o resultado com **um algarismo na casa fracionária**.

Exemplo:

13 e 6 gera uma saída:

```
9,5
```

---

## Q18 

Faça um programa que leia **dois números** e informe **o menor deles**.

Use **apenas operações aritméticas e lógicas**.

---

## Q19 

Em **Edulândia**, o calendário é parecido com o nosso, exceto por:

* **Não existem anos bissextos**
* **Fevereiro possui 30 dias fixos**

Faça um programa que leia **um inteiro representando um mês** do ano Edulandês e imprima **a quantidade de dias desse mês**.

---

## Q20 

Faça um programa que recebe **um inteiro representando um ano** e imprime a **data da Páscoa desse ano**.

Utilize o **algoritmo de Meeus/Jones/Butcher** para o cálculo.

A saída deve estar no formato:

```
dd/mm/aaaa
```

Exemplo:

Entrada:

```
2022
```

Saída:

```
17/04/2022
```
